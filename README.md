# 项目概述 python523

本项目是一个基于Python开发的购物商城系统，采用MVC架构模式，集成了前台和后台管理系统。

## 技术栈

- Python
- Vue
- MySQL
- Django

## 运行环境

- MySQL 5.7
- Python 3.7.7
- 操作系统：Windows 10 / MacOs

## 开发工具

- PyCharm
- 数据库可视化工具：Navicat

## 功能模块

### 前台功能

- 用户注册
- 推荐商品查看
- 商品查看、搜索
- 商品评论、赞、收藏
- 商品在线购买、下单、加入购物车、查看个人购物车
- 在线支付

### 后台功能

- 管理员
  - 个人资料管理
  - 注册用户管理
  - 商品类别管理
  - 商品管理、推荐商品
  - 在线订单管理
  - 在线支付管理
  - 退货申请管理

### 注册用户功能

- 个人资料修改
- 我的订单
- 我的支付查看
- 退货申请、查看申请结果
